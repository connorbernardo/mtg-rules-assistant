# The Command Zone

The command zone is a special game zone used only in Commander (and a few
other variant formats). It holds each player's commander before it enters
the battlefield, and can receive it again later.

Key rules:

- Every player's commander begins the game in the command zone, not in
  their deck or hand.
- A player may cast their commander from the command zone, following
  normal timing rules for when it could otherwise be cast, as long as
  they can pay its cost (including any applicable "commander tax").
- **Commander tax**: each time a player casts their commander from the
  command zone, its cost increases by {2} for each previous time it was
  cast from the command zone that game.
- If a commander would be put into a player's hand, library, or
  graveyard, or would be exiled, that player may choose instead to move
  it to the command zone. This is a replacement effect the player may
  apply, not something that happens automatically.
- This replacement effect can be applied even if the commander changed
  zones because of an opponent's spell or ability.
- A commander in the command zone is not "on the battlefield," so
  effects that only work on permanents currently in play do not apply to
  it while it waits there.

## Common misconception

Players sometimes think a commander automatically returns to the command
zone whenever it would leave the battlefield. It's actually optional —
the controller chooses whether to redirect it there, which matters for
interactions like sacrifice effects or graveyard-based strategies where
a player might *want* their commander to go to the graveyard instead.
